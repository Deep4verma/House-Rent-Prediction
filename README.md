# House Rent Prediction Using Machine Learning

## Project Overview

Developed a machine learning regression model to predict house rental prices based on property characteristics, location details, and housing features. The project helps tenants, property owners, and real estate businesses estimate fair rental values using data-driven insights.

## Objective

Build a predictive model that:

- Estimates house rental prices accurately.
- Identifies factors influencing rental costs.
- Supports real estate pricing decisions.
- Provides insights into housing market trends.

## Dataset

The dataset contains information about residential properties available for rent.

### Features

- Property Size
- Number of Bedrooms
- Number of Bathrooms
- Area Type
- Furnishing Status
- Tenant Preference
- City
- Floor Information
- Property Type

### Target Variable

- Rent

- Python
- Pandas

## Data Preprocessing

- Handled missing values and duplicate records.
- Encoded categorical variables.

Performed detailed analysis to understand rental market trends:

- Rent distribution analysis

## Model Development

Implemented regression models to predict rental prices.

### Models Used

- Linear Regression

## Evaluation Metrics

Model performance was evaluated using:

- R² Score
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)

## Key Insights

- Property size showed a strong influence on rental prices.
- Location significantly affected rent variations.
- Furnished properties generally had higher rental values.
- Number of bedrooms and bathrooms contributed to rental pricing decisions.
